Static one-page website (Arabic RTL) built with **HTML/CSS** + a small JS file.

## Run locally
Just open `index.html` in your browser.

If you want a local server (recommended for some browsers), you can use any static server.

## Images
Put your images into the `assets/` folder.
See `assets/README.md` for the expected filenames.

## What to edit
- Phone / email / address: in `index.html` (Contact section)
- Google Maps embed: replace the iframe URL in the Map section
- Counts: edit the numbers in the Monks section
- Patriarch visits / martyrs / activities: replace placeholder text
